Náplň práce, právomoci a zodpovednosti
Pre nášho klienta hľadáme IT Špecialistu, ktorý bude zodpovedný za správu interných systémov, poskytovanie technickej podpory a zabezpečenie IT infraštruktúry. Hľadáme človeka, ktorý má chuť riešiť technické výzvy a rozvíjať svoje znalosti v oblasti IT.
Náplň práce :
• Riešenie technických problémov na úrovni prvej línie podpory pre hardvér a softvér
• Organizácia a realizácia IT školení pre zamestnancov
• Správa zariadení, ako sú notebooky, tlačiarne či mobilné telefóny
• Inštalácia operačných systémov a aplikácií podľa potrieb spoločnosti
• Evidencia a správa IT majetku vrátane nákupov a skladového hospodárstva
• Manažment mobilných služieb – objednávky, evidencia a nastavenia
• Administrácia interných systémov, najmä SQL databáz, Active Directory a Windows Server
