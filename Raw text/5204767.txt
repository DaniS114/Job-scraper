Team description
Join the team at operational services GmbH within Deutsche Telekom IT Solutions Slovakia, offering opportunities for German- and English-speaking talents. OS is one of the leading ICT service providers in the German market, delivering modern and secure IT solutions, long-term operational support, and ensuring availability of critical business processes for medium-sized and enterprise customers.
Purpose
We are looking for a Senior Database Engineer who will serve as a technical expert for building, monitoring, optimizing, and maintaining MS SQL Server database environments.
The role also includes analyzing datasets, identifying opportunities for optimization, improving performance, and validating technical approaches using data-driven methods.
What will you do?
Administration, operation, and maintenance of Microsoft SQL Server environments
Design, implementation, and optimization of database architectures and data pipelines
Planning and executing installations, upgrades, migrations, and patching of SQL Server
Monitoring performance and tuning SQL Server databases, indexes, and queries
Implementing backup & recovery strategies (Full, Differential, Transactional Log backups)
Troubleshooting database issues, identifying root causes, and implementing corrective actions
Ensuring availability, reliability, and high performance of SQL database systems
Supporting high availability and disaster recovery (HA/DR) configurations
Creating and maintaining system and process documentation
Collaboration with development, infrastructure, and operations teams
Ensuring adherence to security policies, compliance requirements, and audit standards
Contributing to automation initiatives and continuous service improvement
You will succeed if you:
Have a minimum 3 years of hands-on experience with MS SQL Server administration
Understand SQL Server architecture, components, and internal processes
Have practical experience with:
- Installation, configuration, maintenance
- Performance tuning (indexes, execution plans, statistics)
- Backup & recovery (native tools, PITR)
- Monitoring and troubleshooting
Solid knowledge of T-SQL and ability to analyze and optimize complex queries
Experience with Windows Server environment (administration basics required)
Analytical, systematic, and structured working style
Strong teamwork, communication skills, and customer-oriented approach
Ability and willingness to perform on-call duties
Nice-to-Have Technical Knowledge
Oracle
PostgreSQL

High Availability solutions (AlwaysOn, Failover Clustering, Log Shipping, Mirroring)
Command-line / shell scripting
Experience with support tools (e.g., PuTTY, monitoring dashboards, automation tools)
Experience with cloud-based SQL solutions (Azure SQL, SQL Managed Instance – advantage)
Certifications (Preferred)
Microsoft SQL Server Administrator certification (e.g., MCSA, MCSE or current equivalent)
