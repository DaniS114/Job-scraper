Sme žilinská softwerová spoločnosť, ktorá si zakladá na vytváraní pozitívneho pracovného prostredia. Sme hrdí na našu firemnú kultúru a priateľský kolektív. Sme presvedčení, že kvalitné a dobré pracovné vzťahy sú kľúčom k úspechu.

Koho hľadáme?  MS SQL Administrátora/rku
,  ktorého/rej náplň práce bude:
Správa databáz:
Vytváranie a konfigurácia databáz, monitorovanie ich stavu a výkonu. Zabezpečenie dostupnosti integrovaných zálohovacích stratégií.
Optimalizácia výkonu:
Monitorovanie a ladenie výkonu SQL Server databáz. Identifikácia a optimalizácia dotazov a štruktúry databáz pre dosiahnutie najlepšieho možného výkonu.
Zabezpečenie databáz
: Implementácia bezpečnostných politík, prístupových práv a auditovateľnosti. Monitorovanie a reakcia na bezpečnostné hrozby a udalosti.
Podpora vývojového tímu:
Poskytovanie podpory vývojárom pri návrhu databázového modelu a implementácii zmien v databázach.
SQL Development:
Vývoj skriptov, procedúr, funkcií a views.
Požadované zručnosti:
Skúsenosti s inštaláciou, konfiguráciou a správou Microsoft SQL Server.
Hlboká znalosť SQL jazyka a skúsenosti s optimalizáciou dotazov.
Skúsenosti s monitorovaním a ladením výkonu SQL Server databáz.
Dobrá znalosť zálohovacích a obnovovacích procesov.
Schopnosť pracovať s rôznymi nástrojmi pre správu databáz (napr. SQL Server Management Studio, PowerShell).
Dobré porozumenie bezpečnostných princípov a ich aplikácia v kontexte SQL Server databáz.
Výhodou sú skúsenosti s:
Zálohovaním a obnovou databáz
Konfiguráciou mssql replikácie
Správou mssql service broker
Použitím CLR
Budovaním datawarehouse
Memory-Optimized Tables (OLTP)
Change data capture (CDC)
