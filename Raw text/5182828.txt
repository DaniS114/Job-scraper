We are seeking a talented Front End Developer with excellent UX design skills to help build world-class sales tools and map-based solutions. You’ll collaborate with product managers, backend engineers, data experts and users to translate business needs into elegant, user-friendly interfaces that empower our sales and operations teams.
- Design, build, and maintain responsive, high-performance web applications focused on sales enablement and geospatial data visualization.
- Ensure high-quality output with strong attention to detail, translating product vision into the final user experience.
- Work closely with product management and the user community to create engaging interfaces, interactive maps, and workflow-driven dashboards.
- Connect front-end applications to backend APIs and data sources to deliver accurate, real-time, and batch data displays.
- Optimize web applications for speed, scalability, and accessibility across multiple devices and browsers.
- Transform business requirements and user feedback into thoughtful, intuitive design enhancements.
- Participate in code reviews, maintain documentation, and follow best practices to support continuous improvement.
- Promote a culture of creativity, learning, and ongoing development within the team.
Požadujeme:
Required:
- 5+ years in front-end web development with JavaScript, HTML5, CSS3, and modern frameworks (React, Angular, Vue).
- Experience delivering successful products with creative and strategic thinking.
- Familiarity with AI-driven tools (GitHub Copilot, LLMs, RAGs) and emerging technologies.
- Strong software engineering practices: code quality, testing, DevOps.
- Experience with interactive maps (Leaflet, Azure Maps) and responsive UX design.
- Knowledge of RESTful APIs, asynchronous data flows, and version control (Git/GitHub).
- Experience with containerized apps, Microsoft Azure Container Apps, and IIS hosting.
Nice to Have:
- Telecom or sales enablement experience.
- Data visualization libraries (D3.js, Chart.js).
- Agile methodology exposure.
- Geographic data stores (GeoPandas, DBF).
- Backend knowledge (Python, Flask, FastAPI).
