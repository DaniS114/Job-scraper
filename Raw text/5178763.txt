Náplň práce, právomoci a zodpovednosti
Budujeme modernú aplikáciu pre verejný sektor, ktorá spája mobilné a webové prostredie do jedného celku. Využívame React Native, Expo a React Native Web, aby sme priniesli konzistentný používateľský zážitok na všetkých platformách. Na dlhodobú udržateľnosť a škálovanie používame Turborepo monorepo, architektúru staviame na Zustand pre správu stavu a React Query pre efektívnu prácu s dátami. Aplikácia je pripravená fungovať aj offline – vďaka SQLite databáze.
Zodpovednosti:
Navrhovať a vyvíjať funkcie v React Native + Expo.
Riešiť architektonické otázky pri práci s React Navigation, React Query, Zustand, SQLite.
Vykonávať code review, mentorovať juniorov a dohliadať na kvalitu kódu.
Podieľať sa na implementácii offline-first prístupu a práce s lokálnou databázou.
