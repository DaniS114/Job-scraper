Frontend Software Engineer
Place of work
Bratislava, Slovakia (Job with occasional home office)
Wage (gross)
From 3 000 EUR/month
* Financial bonus based on performance paid twice a year up to 10% from the base salary paid during the evaluation period (normally 6 months).

* The final basic wage component can be increased accordingly to individual skills and experience of the selected candidate.
Contract type
full-time
We’re looking for an experienced frontend developer who thrives in a dynamic environment, enjoys building scalable systems, and is passionate about delivering high-quality digital products used by millions.
You’ll join a cross-functional team of engineers working on high-impact global websites like eset.com, esetscienceaward.sk, welivesecurity.com, support.eset.com, and others. We build websites for visitors around the globe (200,000 visitors per day) and our mission is to educate them on how to protect themselves against cybersecurity threats.
What you’ll do:
Develop and maintain features across a multi-project, highly scalable CMS system used across ESET’s global web platforms.
Write modern, clean, and efficient code in Vue (Nuxt) or React (Next), ideally both.
Build reusable UI components and collaborate on a shared design system.
Integrate AI tools like custom assistants and chatbots to enhance user experience and internal workflows.
Work closely with backend engineers to integrate REST/GraphQL APIs with security and performance in mind.
Participate in Agile development: daily standups, sprint planning, reviews, and retrospectives.
Contribute to code reviews, technical discussions, and architectural decisions.
This position is a maternity leave cover.
What we’re looking for:
5+ years of professional experience in frontend development.
Solid proficiency in Vue.js (Nuxt) and React (Next).
Experience with component-based architecture and scalable codebases.
Familiarity with headless CMS platforms and building modular frontend apps.
Understanding of web performance, SEO best practices, and accessibility.
Experience with TypeScript, build tools (Vite, Webpack), and modern JavaScript patterns.
Strong GIT skills and familiarity with CI/CD workflows.
Comfortable collaborating across teams, taking ownership, and working independently.
Interested in emerging technologies and excited to help push the boundaries of ESET’s websites.
Good English communication skills (B2+ level).
Bonus points for:
Experience integrating or building AI-based solutions (chatbots, agents, MCP servers,..).
Familiarity with Docker and cloud-based development environments.
Background in creating design systems or frontend architecture for multi-brand projects.
Experience with PHP & Laravel framework.
Familiarity with automated component and unit testing tools like Vitest or Jest.
What we offer:
A chance to shape global websites and highly-scalable CMS solution.
Work on cutting-edge projects.
Tech stack includes Typescript, Vue (Nuxt), React (Next), GraphQL, Vite, and more.
Access to premium AI tools: GitHub Copilot, MS Copilot, custom internal AI tooling, etc.
Smart, motivated colleagues and a supportive learning culture.
Clear growth path
