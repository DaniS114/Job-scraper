POSITION
DESCRIPTION
We’re looking for a Kubernetes Engineer who’s passionate about cloud-native tech and automation. In this role, you’ll design, deploy, and manage containerized environments using Kubernetes, CI/CD pipelines, and Infrastructure as Code tools.
WHAT WILL
YOU DO
•	Containerize existing and new applications, tools, and websites and move them to Kubernetes
•	Engineer solutions on major cloud platforms
•	Implement scalable and serverless infrastructure using latest available technologies
•	Work with state-of-the-art technologies and from every aspect of the DevOps/Cloud Ecosystem from development through deployment to operations and monitoring
•	Develop health monitoring/metrics for status of the Kubernetes cluster
•	Work in conjunction with IT, engineering, and business groups to understand functionality, scalability, performance, security, and integration requirements
WHAT WE
EXPECT
•	Hands-on (3-5 years) experience in developing, deploying, tuning and debugging applications on Kubernetes
•	Experience either in developing code in at least one high-level programming language or in building automated infrastructures
•	Familiar with cloud platform administration and/or development
•	Fundamental knowledge of key DevOps concepts – continuous integration and delivery, Infrastructure As Code and Cloud concept (IaaS, PaaS, SaaS, FaaS)
•	Demonstrated experience in one or more of the following technologies and platforms: Kubernetes, Docker, PowerShell, Bash, Python, etc.
•	University educated (Bachelor’s or Master’s degree in IT or a related field)
•	Conversational level of English language (B2 level and above)
•	Good understanding of German language, with active communication as well at least in a written form (B1-B2 level and above)
•	Fluent in Slovak (C2 level)
