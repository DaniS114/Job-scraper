Informácie o pracovnom mieste
Náplň práce, právomoci a zodpovednosti
Čo budeš robiť
• Infrastructure & DevOps
• budovať a prevádzkovať infraštruktúru ako kód (Terraform, Terragrunt, Helm, Ansible) v Azure • nastavovať siete a bezpečnostné prvky v štýle “default deny” (NSG/ASG, Azure Firewall/WAF, segmentácia).
• nasadzovať a spravovať VPN/IPsec (site-to-site, point-to-site), routing a vysokú dostupnosť.
• zavádzať policy-as-code (Azure Policy, OPA), správu secrets (HashiCorp Vault, Key Vault) a bezpečné CI/CD (Azure DevOps/GitHub Actions)
• monitorovať infraštruktúru a bezpečnosť pomocou Azure Monitor, Log Analytics, Sentinel, Splunk
• tvoriť workflows pre incidenty, zálohy/DR a patch management
• vykonávať infraštrukturné audity a pentesty
Stack, s ktorým budeš robiť:
• cloud & infra: Azure, Terraform, Terragrunt, Azure Policy, Key Vault, Defender for Cloud, Sentinel
• devops & ci/cd: Azure DevOps, GitHub Actions, Git, Helm, Ansible
• os & networking: Linux, VPN/IPsec, DNS, PKI
• security & monitoring: Azure Monitor, Log Analytics, Splunk, WAF, Firewall, audit/skenovacie nástroje
• containers: Docker, Azure Container Registry
