DESCRIPTION
Are you passionate about AI and eager to shape the future of machine learning on the Azure platform? Join our dynamic Accenture Business Applications team, where you’ll collaborate with global experts to deliver innovative solutions in Natural Language Processing, speech and image recognition. If you thrive on tackling real-world challenges and want to make a tangible impact across industries, we want to meet you.
WHAT WILL
YOU DO
• Continuous development of our clients’ machine learning platform and products based on mainly Azure
• Identifying, creating and preparing data required for machine learning algorithms
• Building virtual assistants, recommendation engines and different tasks using computer vision based on client requirements
• Deploying and maintaining machine learning solutions in production
• Using Agile/SCRUM methodology
• Depending on your experience you might work on custom machine learning solution architecture, design, definition and implementation
WHAT WE
EXPECT
• 5+ years of experience with building applications in Python
• Solid background in Python scientific computing libraries
• Proven track record in Natural Language Processing (NLP) and Computer Vision, including projects in object detection and classification
• Familiar with machine learning libraries like Tensorflow, Keras, PyTorch and similar
• Bachelor’s or Master’s degree in IT or a related field
• English proficiency at Upper intermediate (B2) or Advanced (C1), German language is nice to have
Nice to have:
• Experience with Azure cloud technologies, Azure Machine Learning
• Familiar with MS Bot Framework
