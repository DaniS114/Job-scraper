Náplň práce, právomoci a zodpovednosti
Si Backend Developer, ktorý sa nezľakne žiadneho kódu?
Máš rád výzvy, stabilný tím a projekty, kde sa vieš posunúť dopredu?
Hľadáš prostredie, kde sa pracuje s modernými technológiami a kde máš možnosť rásť nielen technicky?
Potom hľadáme práve Teba.
Čo ťa u nás čaká:
vývoj moderných aplikácií      v Jave (s využitím Spring Boot a ďalších Spring frameworkov),
nasadzovanie do cloudových      a orchestračných platforiem (Kubernetes, OpenShift, Docker, …),
plná automatizácia pomocou      CI/CD nástrojov,
dôraz na kvalitu kódu –      Unit testy, code review, SonarQ,
práca na projektoch pre      rôznych klientov – od enterprise riešení až po moderné digitálne      platformy,
tím, ktorý ťa podrží a kde      sa môžeš učiť aj zdieľať skúsenosti.
Čo od teba očakávame:
skúsenosti s Java &      Spring Boot,
chuť učiť sa nové      technológie a prístupy,
výhodou je znalosť CI/CD,      kontajnerizácie alebo cloudových prostredí.
Čo ti ponúkame:
stabilitu inovatívnej      slovenskej spoločnosti,
možnosť podieľať sa na      zaujímavých projektoch pre lokálnych aj zahraničných klientov,
priestor pre odborný rast      a vzdelávanie,
priateľskú firemnú kultúru      a tím, ktorý ťa nenechá v štichu.
Aké ználosti u nás využiješ?
skúsenosti s Java      (8/11/17+) a Spring Boot (+ Spring Data, Spring Security, Spring Cloud),
znalosť REST API (tvorba a      integrácia),
základy SQL a relačných      databáz (PostgreSQL, Oracle, MS SQL), výhodou aj NoSQL (MongoDB, Redis),
práca s verzionovacím      systémom Git (GitLab/GitHub/Bitbucket)
Výhodou sú:
znalosť CI/CD nástrojov      (Jenkins, GitLab CI, Azure DevOps…),
skúsenosti s      kontajnerizáciou a orchestráciou (Docker, Kubernetes, OpenShift),
orientácia v testovaní      (JUnit, Mockito),
skúsenosti s Maven/Gradle,      cloudovým prostredím (AWS, Azure, GCP) a mikroslužbovou architektúrou.
