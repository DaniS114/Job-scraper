Informácie o pracovnom mieste
Náplň práce, právomoci a zodpovednosti
The Backend Developer is part of the SW Development Team and is responsible for developing software for local and cloud communication with embedded devices.
• Working in a software development team
• Software architecture, design and development of C/C++, .Net applications
• Contributing into the KiConnect Platform, a Kistler IoT communication platform based on the OCF specification
• Analysis of requirements and design, implementing and testing software modules
• Elaborate software requirements specifications for given topics
• Taking responsibility for parts of the project (module, components)
• Bugfixing
• Supporting the team by creating professional software architecture and design
• Applying Kistler methodology for professional software development
• Leading new coworkers regarding the use of technology and methods (teaching, coaching)
